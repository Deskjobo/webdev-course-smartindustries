Hola a todos. Para este tutorial ya deon aprender lo que es un sitio web, la estructura de HTML, sus elementos básicos y cómo darle estilo con CSS. Ahora vamos a poner todo esto en prueba y crear nuestra primera página web completa. El resultado final va a ser algo así como una página de perfil, una entrada de blog o alguna publicación simple. Para este tutorial no es necesario instalar nada. Usaremos la herramienta gratuita de Codepen, la pueden encontrar en este enlace, codepen.io. Y una vez que se en esta página, hagamos clic en prueba si es que no quieren crearse una cuenta. Y verás tres secciones: HTML, CSS y JS. No se asusten con JS, solo vamos a utilizar lo que hemos aprendido hasta ahora. Entonces, déjame minimizar este editor y comencemos con HTML, que es la estructura o el esqueleto de nuestra página. Entonces, para un título voy a usar la etiqueta H1. Pongámosle mi primera página. Y acuérdense siempre de cerrar las etiquetas. Más abajo podemos tal vez poner una imagen. Utilizamos CRS para darle una fuente a nuestra imagen. Por ahora lo voy a dejar vacío, pero más adelante pondré el enlace a una imagen. Alt ayuda a poner el texto alternativo, que más adelante explicaremos más para qué se usa, y algún texto escritivo simple como foto. Bueno, continuemos. Un subtítulo podría ser con la etiqueta H2. Hola mundo. Siempre acuérdense de cerrarlo con la etiqueta, la misma etiqueta que usaron para abrirlo. Y por último, pongamos un poco de texto simple con la etiqueta de P, que significa como párrafo. Bienvenidos a mi página web. La diferencia entre una etiqueta de apertura y una etiqueta de cerradura siempre va a ser este slash para el frente. Entonces, no olviden que si tienes dos etiquetas, la última te llevar este slash. Bueno, si ven aquí el resultado, nuestra página ya se está formando. Tenemos un título, una imagen que aún no lo hemos puesto, un subtítulo y un párrafo. Ya es una página web, aunque todavía no se ve muy bien eso lo vamos a cambiar con CSS, que es lo que nos va a dar estilo. Entonces, vayamos al panel CSS y empecemos a poner un poco de color a nuestra página abriendo la sección de body, que es para todas las clases que sean del cuerpo de nuestra página. y le podemos agregar una fuente, por ejemplo, font family es la familia de la fuente Arial. Sería de esta manera. Sif. En CSS todas las líneas se deben cerrar con un punto y coma. Podríamos ponerle un fondo. En inglés se dice background y color dos puntos. algún gris oscuro podría ser esta hexadecimal, que quiere decir que son niveles de rojo, azul y verde. Acordémonos de cerrar la línea. Podríamos alinear el texto para que se vea más ordenado con text align o alinear texto en inglés. Center quiere decir que se va a alinear el centro de la pantalla. Y por último, un poco de espaciado a los lados. En inglés se dice padding 50 unidades de píxeles. Veamos cómo se ve. Efectivamente, ahora está alineado, tiene espacio y ha cambiado la fuente a una más agradable. Podemos continuar, por ejemplo, a todas nuestrasquitas H1 agregarle un color principal para los títulos. color dos puntos. Bueno, le puse este color para que se pueda notar un poco más la diferencia. Veamos, es la imagen que tiene nuestra etiqueta img. Abramos las llaves y podríamos ponerle un borde, por ejemplo, con un radio para que sea redondeado. Radius en inglés del 50%. Y un borde de tres píxeles. En el caso de borde se necesitan tres distintos atributos. El en el caso de bordo se necesor que se expresa en PCLes, sólido para la transparencia y el color que pongámoslo del mismo color que pusimos para tener consistencia. Y ahí se ve el borde. Claro que será mejor cuando tengamos una imagen adentro. Bueno, por último, para ser completos, pongámosle un estilo al párrafo. H podría ser, por ejemplo, una un ancho máximo, que en inglés se diría max withd píxeles o 400, perdón. Ahora si tenemos mucho texto no se va a pasar de estos márgenes. Además de eso, pongámosle un color un poco distinto al negro. Y listo. Ahí agregué un enlace para la imagen y le cambié algunos colores para que sea más atractivo y con eso tendrías su primera página web.
